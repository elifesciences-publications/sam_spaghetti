============
Installation
============

-----------------
Conda environment
-----------------

Creating a new environment (recommended)
----------------------------------------

.. code-block::

    conda create -n sam_spaghetti -c mosaic python>=3.6 sam_spaghetti


In an existing Python 3.6+ environment
--------------------------------------


.. code-block::

    conda install -c mosaic sam_spaghetti
